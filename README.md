## 💻 Fundamentos de Linux y Sistemas
* **Kernel**: El núcleo del sistema operativo.
* **Shell**: La terminal donde ejecuto mis instrucciones.
* **Root**: Superusuario con poder total sobre el sistema.

## 🌐 Redes (Networking)
* **IP Address**: Dirección única de un equipo en la red.
* **Subnet Mask**: Máscara de subred.
* **Default Gateway**: El router que nos conecta a internet.

## 🔍 Ciberseguridad y Criminología
* **Vulnerability**: Una debilidad o fallo en un sistema.
* **Encryption**: Cifrado de datos para protección.
* **Payload**: La carga útil que ejecuta la acción de un ataque o prueba.
* **Phishing**: Estafa para suplantar identidades.
* **Hardening**: Proceso de fortalecer un sistema.## 🛠️ Herramientas de Diagnóstico de Red

* **ip a**: Utilizado para identificar las interfaces y direcciones IP de mis máquinas.
* **ping -c 4**: Utilizado para verificar la latencia y conectividad entre Ubuntu (base) y Kali (auditoría). Resultado: 0% pérdida de paquetes.
